## 03.1 크기 단위 

|단위|설명|
|:-:|:-:|
|%|백분율 단위|
|em|배수 단위|
|px|픽셀|

```html
<!DOCTYPE html>
<html>
<head>
    <title>CSS3 Style Property Basic</title>
    <Style>
        /* 퍼센트 단위는 100%가 초기에 설정된 크기 */
        p:nth-child(1) { }
        p:nth-child(2) {font-size: 100%;}
        p:nth-child(3) {font-size: 150%;}
        p:nth-child(4) {font-size: 200%;}

        /* em 단위는 1배=1em=100%이며 1.5배=1.5em=150%이다. */
        /* p:nth-child(1) { }
        p:nth-child(2) {font-size: 1.0em;}
        p:nth-child(3) {font-size: 1.5em;}
        p:nth-child(4) {font-size: 2.0em;} */
        
        /* 절대적 크기를 지정할 때 px 단위 사용(기본 font-size 속성이 16px) */
        /* p:nth-child(1) { }
        p:nth-child(2) {font-size: 16px;}
        p:nth-child(3) {font-size: 24px;}
        p:nth-child(4) {font-size: 32px;} */
    </Style>
</head>
<body>
    <p>Hello</p>
    <p>Hello</p>
    <p>Hello</p>
    <p>Hello</p>
</body>
</html>
```
